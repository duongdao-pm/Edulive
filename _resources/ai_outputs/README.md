# AI Outputs — Shared Folder

Folder dùng chung giữa các team, để anh Ngọc và cross-team xem output AI.

## Mục đích
- Lưu output kho ảnh KNTT (Nam, Đạt) → chị Ánh + anh Ngọc review
- Lưu kết quả chấm điểm LLM (khi có)
- Lưu planning/milestones AI team

## Quy tắc đặt tên file
```
[Task ID]_[mô tả ngắn].[ext]
Ví dụ:
  EDU002-AI-002_anh_toan_lop4_tap1.zip
  EDU002-AI-003_planning_milestones.md
  EDU002-AI-001_tool_bai_giang_spec.md
```

## Ai có quyền
- **Ghi**: AI Team (Thịnh, Tuyến, Đạt, Nam)
- **Đọc**: Tất cả team + PM + anh Ngọc
