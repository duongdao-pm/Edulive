# Resources — Edulive (chung)

Thư mục chứa file & tài liệu dùng chung cho toàn bộ workspace, không thuộc riêng dự án nào.

## Đưa gì vào đây?
- Logo, brand assets
- Template chung (hợp đồng, báo cáo, email mẫu)
- Tài liệu công ty (quy trình, policy, org chart)
- Thông tin đối tác / khách hàng chung
- File config, credentials (cẩn thận bảo mật)
- Bất kỳ resource nào nhiều dự án cùng cần

## Quy ước đặt tên
```
[loại]_[mô-tả-ngắn].[ext]
VD: template_brd_v1.docx
    brand_logo_edulive.png
    policy_deploy_checklist.pdf
```

## Lưu ý
- File ở đây là **read-only** đối với agents — không sửa, không xóa
- Agents tham chiếu: `_resources/ten-file.ext` (từ root Edulive)
